Mise en pratique

L'objectif de ces travaux pratiques est de vous introduire aux principes de l'éco-conception numérique, en appliquant ces concepts à un cas réel : le site web de l'association étudiante ESIROI. À travers ces TPs, vous apprendrez à analyser et optimiser les performances des sites web tout en réduisant leur empreinte écologique.

Prérequis : Connaissances de base en HTML, CSS, et JavaScript, ainsi que de l'expérience avec Git, Docker, et Docker Compose. Vous devez disposer d'un environnement de développement sous Ubuntu, Windows, ou macOS.

Le site web de l'Association Étudiante ESIROI a été conçu pour offrir une plateforme dynamique aux étudiants, leur permettant d'accéder facilement aux informations relatives aux activités, événements et services proposés par l'association.
Aujourd'hui, l'association souhaite améliorer son site en intégrant les principes d'éco-conception numérique afin de réduire son impact environnemental tout en maintenant une expérience utilisateur de qualité. Ce projet servira de base pour nos travaux pratiques, soulignant l'importance de l'éco-responsabilité dans le développement web.

Découvrez le site de l'association.

Avant de commencer les TPs, veuillez répondre à ce QCM sur les questions générales pour vous préparer.

TP 01 : Introduction à SonarQube et Eco Code

Installez et configurez SonarQube avec Eco Code pour analyser un projet et identifier les améliorations possibles en termes d'éco-conception..

TP 02: Optimisation des Ressources Web

Optimisez les ressources du site du BDE pour réduire son **empreinte écologique** et améliorer ses performances.

TP 03: Détection et Réduction des Scripts Inutilisés

Identifiez et éliminez les scripts JavaScript inutilisés ou redondants pour un site plus performant et éco-responsable.

TP 04: Optimisation de l'Utilisation des Polices Web

Apprenez à optimiser les polices web pour améliorer les performances et réduire l'empreinte écologique du site.
